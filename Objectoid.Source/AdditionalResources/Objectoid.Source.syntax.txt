Tokens:
* Keyword     Keywords start with @
* Numeric     Numeric values start with a digit or a minus sign
* String      Strings start with a quotation mark
* Constant    Constants start with a letter





Special Characters:
#     Start of comment
/     Indication to parse the next row along with the current row
:     End of parse (useful for placing multiple parse blocks within the same row)
\     Escape character for strings





Escape Sequences:
Sequence     Description                 Unicode
\'           Single quote                0x0027
\" 	         Double quote                0x0022
\\ 	         Backslash                   0x005C
\0 	         Null                        0x0000
\a 	         Alert                       0x0007
\b 	         Backspace                   0x0008
\e 	         Escape                      0x001B
\f 	         Form feed                   0x000C
\n 	         New line                    0x000A
\r 	         Carriage return             0x000D
\t 	         Horizontal tab              0x0009
\v 	         Vertical tab                0x000B
\u 	         Unicode escape sequence     0xHHHH (example: \u00F1 = "ñ")





Header:

Keywords:
* @ProtocolPrefix     Declaration of a protocol prefix     @ProtocolPrefix <String>




Protocol Prefix:

Prefix to use when determining a protocol for an import block

Example:
 @ProtocolPrefix "Skin."
 @Object
 "summer skin" @Import "Summer"
 @EndImport
 "winter skin" @Import "Winter"
 @EndImport
 @EndObject
Is the same as
 @Object
 "summer skin" @Import "Skin.Summer"
 @EndImport
 "winter skin" @Import "Skin.Winter"
 @EndImport
 @EndObject

Remarks:
Multiple Protocol Prefix statements can be used





Element:

Keywords:
* @String        Declaration of string value                              @String <String> [<String> [<String> ... ]]
* @NTString      Declaration of null-terminated string value              @NTString <String>
* @Bool          Declaration of boolean value                             @Bool <Constant>
* @UInt8         Declaration of 8-bit unsigned value                      @UInt8 <Numeric>
* @UInt16        Declaration of 16-bit unsigned value                     @UInt16 <Numeric>
* @UInt32        Declaration of 32-bit unsigned value                     @UInt32 <Numeric>
* @UInt64        Declaration of 64-bit unsigned value                     @UInt64 <Numeric>
* @Int8          Declaration of 8-bit unsigned value                      @Int8 <Numeric>
* @Int16         Declaration of 16-bit unsigned value                     @Int16 <Numeric>
* @Int32         Declaration of 32-bit unsigned value                     @Int32 <Numeric>
* @Int64         Declaration of 64-bit unsigned value                     @Int64 <Numeric>
* @Single        Declaration of single-precision floating-point value     @Single <Numeric>
* @Double        Declaration of double-precision floating-point value     @Double <Numeric>
* @RawBytes      Declaration of an array of raw bytes                     @RawBytes <Numeric> [<Numeric> [<Numeric> ... ]]
* @Null          Null declaration                                         @Null
* @Object        Declaration of an object block                           @Object
* @List          Declaration of a list block                              @List
* @Import        Declaration of an import block                           @Import <String>
* @EndObject     End of an object block                                   @EndObject
* @EndList       End of an list block                                     @EndList
* @EndImport     End of an import block                                   @EndImport

Object example:
@Object
"First Name" @String "John"
"Last Name" @String "Doe"
"Age" @UInt8 32
@EndObject

List example:
@List
@String "Zero"
@String "One"
@String "Two"
@String "Three"
@String "Four"
@EndList

Import example:
@Import "Image"
"Path" @String "C:\\path\\to\\image.bmp"
"Type" @String "Bitmap"
@EndImport





Example:
@ProtocolPrefix "Skin"
@Object
	"First Name" @String "John"
	"Last Name" @String "Doe"
	"Age" @UInt8 75
	"Alive" @Bool true
	"Spouse" @String "Jane"
	"Children" @List
		@String "Jack"
		@String "Jill"
	@EndList
@EndObject
